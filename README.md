# Housing price prediction
This repository contains the notebooks used to study the housing price predictions in Kaggle

## Dataset
The dataset used in this experiment is the California Housing Prices CSV file. The data contains information from the 1990 California census. This dataset is a modified version of the California Housing dataset available from: [Lu√≠s Torgo's page](https://www.dcc.fc.up.pt/~ltorgo/Regression/cal_housing.html).

## Data description
The data pertains to the houses found in a given California district and some summary stats about them based on the 1990 census data. The columns are as follows, their names are pretty self explanitory:

longitude
latitude
housingmedianage
total_rooms
total_bedrooms
population
households
median_income
medianhousevalue
ocean_proximity
